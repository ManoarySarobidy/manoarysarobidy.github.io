## ERP Software : Logiciel de gestion des ressources d'entreprise

Ce projet consiste en un **logiciel ERP (Enterprise Resource Planning)** conçu pour centraliser et automatiser les processus de gestion au sein d'une entreprise. Il offre une plateforme intégrée pour gérer divers modules essentiels, tels que les finances, les ressources humaines, la gestion des stocks et les ventes.

### Fonctionnalités principales
- **Gestion des finances** : Suivi des budgets, gestion des factures et génération de rapports financiers.
- **Ressources humaines** : Gestion des employés, suivi des présences et traitement des salaires avec leurs primes, la gestion des congés.
- **Gestion des stocks** : Contrôle des inventaires, gestion des fournisseurs et alertes pour les seuils critiques.
- **Module des ventes** : Gestion des commandes, suivi des clients et rapports de performances.
- **Tableaux de bord** : Vue d'ensemble en temps réel avec des KPI (Indicateurs clés de performance).

### Stack technologique
- **Langage de programmation** : Java, Python, Javascript
- **Frameworks** : Spring Boot, Flask pour les apis de statistiques, JQuery.js pour une interaction utilisateur conviviale
- **Base de données** : PostgreSQL, Oracle
- **Outils complémentaires** : Docker pour le déploiement, Git pour le versioning, Visual Studio Code pour le développement
